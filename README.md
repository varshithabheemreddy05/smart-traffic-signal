# 🚦 Smart Traffic Signal System

## 📌 Overview
This project simulates a smart traffic signal system that dynamically assigns green signals based on traffic density.

## 🚀 Features
- Detects highest traffic lane
- Handles tie situations 
- Uses round-robin rotation for fairness
- Simulates real-time traffic decisions

## 🧑‍💻 Technologies
- Python
